n8n은 다양한 서비스와 API를 연결해 반복 작업을 자동을 워크플로우 자동화 플랫폼이다.

다양한 자동화 플랫폼 중에서도 오픈소스 기반 무료이고 코드와 노코드 모두 제공하면서 드래그 앤 드롭 인터페이스로 쉽게 시작할 수 있따는 장점 그리고 AI 기능을 대폭 강화하면서 AI Agent Tool 노드 추가, Claude 3.5 및 Gemini 2.5 Pro 직접 연동으로 인공지능 자동화 워크플로우를 쉽게 구축할 수 있음
또 셀프호스팅이 가능해서 내 서버에 직접 설치해 완전한 데이터 통제권도 확보가능 앱연동도 지원해서 API 연결 , 데이터베이스, 서비스 등 다양한 통합도 제공 이런장점으로 주목받는 자동화 서비스다

## N8n에서 트리거 노드란
워크플로우의 첫 번째 노드로, 워크플로우가 언제, 어떤 조건에서 실행될지 결정함. 
즉 트리거가 시작되어야 워크플로우가 시작됨 

### 트리거 노드의 3가지 핵심 역할
1. 시작점 역할
   - 워크플로우의 첫 번째 노드로서 모든 작업 흐름이 여기서부터 시작됨
2. 조건 설정
   - 워크플로우가 언제, 어떤 상황에서 실행될지 결정함
3. 데이터 제공 
   - 워크플로우에 필요한 초기 데이터를 수집하고 전달 

### 트리거 노드 4가지 유형
1. 수동 트리거
   - 내가 직접 실행
     - 버튼 누르기
2. 시간 트리거
   - 정해진 시간에 자동 실행
     - 특정 시간 지정
3. 이벤트 트리거
   - 특정 이벤트 발생 시 실행
     - 외부 이벤트(웹훅, 이메일, 폼 제출 등)에 반응
     - 웹훅이란?
       - "인터넷 벨"이라고 생각하면 됨. 
       - 예를 들어 고객이 문의 양식을 작성하면 -> 자동으로 처리 등 
      ![웹훅트리거]({{ '/assets/images/n8n/웹훅트리거.png' | relative_url }})
4. 시스템 트리거
   - 시스템 상태 변화 시 실행
     - 서버 시작, 오류 발생, 데이터베이스 변경 등에 반응 


### 챗 트리거
구글 재미나이 모델과 AI Agent를 연결해서 대화하고 내용을 DB에 저장
- 고객 서비스 챗봇
  - 고객 문의를 자연어로 접수하고 적절한 워크플로우 실행
- 내부 업무 지원 어시스턴트
  - 팀원들의 일상 업무를 대화형 인터페이스로 자동화
- 대화형 데이터 조회 시스템
  - 복잡한 데이터 쿼리를 자연어 대화로 실행 
![폼트리거]({{ '/assets/images/n8n/n8n챗.png' | relative_url }})

### 폼 트리거
웹 폼을 통해 데이터를 수집하고 자동화 워크플로우를 실행
- 모바일 최적화
  - 모든 디파이스에서 완벽하게 작동하는 폼 디자인으로 사용자 경험 개선
- 스마트 폼 로직
  - 사용자 응답에 따라 다른 질문을 표시하는 동적 폼 생성
- 풍부한 입력 옵션
  - 다양한 형태의 데이터를 수집할 수 있는 입력 필드 지원 
![폼트리거]({{ '/assets/images/n8n/n8n폼트리거.png' | relative_url }})

### 이메일 트리거
이메일이 도착하면 자동으로 워크플로우가 실행됨
- 이메일 기반 자동화
  - 메일이 도착하면 워크플로우 시작
- IMAP 설정 방법
  - Email Trigger(IMAP) 노드 추가
  - 이메일 계정  인증 정보 입력
  - 확인 주기 및 폴더 생성
- 업무 자동 분류 예시
  - 실제 활용 시나리오 


## 실무 시나리오별 추천 트리거
- **데이터 백업 자동화**
  - Schedule Trigger : 서버 부하가 적은 시간대를 선택해 정기적인 데이터 백업
- **고객 문의 처리**
  - Form + Email Trigger : 웹사이트 문의와 이메일 문의를 한 워크플로우로 통합
- **팀 내부 업무 지원**
  - Chat Trigger : 팀원들이 자연어로 데이터 조회, 보고서 생성 등을 요청할 수 있게 설계 
- **외부 시스템 연동**
  - Webhook Trigger : 결제 시스템, CRM, 마케팅 툴 등 외부 서비스와 연동할 때
- **보고서 생성**
  - Schedule + Manual Trigger : 정기 보고서는 자동으로, 필요 시 수동으로도 생성 가능하게 설정.  
- **알림 시스템**
  - Email Webhook Trigger : 중요 이메일이나 시스템 알림을 받으면 자동으로 모바일이나 슬랙 등으로 전달하는 워크플로우 구성 가능
